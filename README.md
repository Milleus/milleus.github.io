# Milleus.github.io

Milleus.github.io is my personal website using Github Pages with Jekyll.

## How to run locally?

1. Install Ruby.
2. Install Bundler and Jekyll 3.8.5, see [dependencies](https://pages.github.com/versions/).
3. Install gem packages with `bundle install`.
4. Go to folder in terminal and run command `bundle exec jekyll serve`.
5. View website at `http://127.0.0.1:4000`.
